MiniShop is a lightweight e-commerce web application built to demonstrate core online shopping functionalities using modern frontend tooling and a clean, scalable project structure. This module is part of the **mini-shop_E-commerce** repository.

---

## Project Overview

MiniShop provides a foundational e-commerce experience, focusing on simplicity, maintainability, and responsiveness. The project is suitable for learning purposes, portfolio demonstration, and future extension into a full-scale production system.

---

## Features

- Responsive product listing interface  
- Product detail view  
- Add and remove items from shopping cart  
- Cart state management  
- Basic checkout workflow  
- Mobile-first responsive design  

---

## Technology Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Build Tool:** Vite  
- **Styling:** Tailwind CSS  
- **Package Manager:** npm  
- **Version Control:** Git
